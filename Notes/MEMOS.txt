## GNUstep Installation dir

Installing an app for ALL USERS:

export LOCAL_INSTALL_DIR=$(gnustep-config --variable=GNUSTEP_LOCAL_APPS)
sudo -E make install

if no target install:
sudo cp ${APP_NAME}.app $LOCAL_INSTALL_DIR/
