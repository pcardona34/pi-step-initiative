OTHER THEMES
------------
GSPISIN.tar.gz => PISIN.theme: it is the main GNUstep theme of PiSiN Desktop.
Most of the icons were reworked from original Papirus Icons set to the 48x48 appearance and TIFF format.

WMPISIN.tar.gz => PISIN.themed: color theme for Window Maker.

FocusEnchantement2.fwtz => theme for FocusWriter (see Writer.app wrapper)
